# ANDROID RSS APP with Node.js REST API

## Run application
```
git clone https://github.com/PawlakPiotr/RSS-Android-App.git
```

### REST Api

```
cd ./api
npm install
```

After installing dependencies:

```
node server.js
```

Note: MongoDb required
   

### Android app

1. Open Android Studio
2. Build project
3. Start emulator
